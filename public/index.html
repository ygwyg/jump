intent api

Describe what you want in plain English. jump(el, "fade in slide up") — Jump resolves the keyframes, easing, and timing automatically. An AI generates this correctly on the first try. Combine any two intents with a space for compound animations.

spring physics

Jump solves the damped harmonic oscillator and encodes the result as a CSS linear() easing — real physics running on the compositor thread, not a JS animation loop. Click any ball to feel the spring.

gentle
default
bouncy

wobbly
// presets — property getters, not function calls jump(el, "enter", springs.bouncy) // real overshoot jump(el, "enter", springs.wobbly) // oscillates // or custom physics const s = createSpring({ stiffness: 200, damping: 18 }) jump(el, "enter from bottom", s)
stagger

Pass any selector, array, or NodeList. Add a stagger value. Each element gets a calculated delay offset — no loops needed. Grid ripple from center, eased distribution, all built in.

// grid ripple from center jump(".dot", "enter from bottom", { stagger: stagger(40, { grid: [8, 4], from: "center" }) })
sequence

jump.sequence() queues steps in order. jump.parallel() fires them all at once. Both return controls with play, pause, seek, and a finished promise.

gestures

Touch-safe hover, keyboard-accessible press, drag with momentum and spring snap-back. Each attaches in one line and cleans up on stop().

jump.hover(el, { onEnter: el => jump.to(el, { scale: 1.06 }, springs.stiff), onLeave: el => jump.to(el, { scale: 1 }, springs.gentle) }) jump.drag(el, { bounds: { left: -80, right: 80 } })

layout / flip

Capture before, mutate, flip after. Every element glides to its new position. Shared element transitions with a single string ID — thumbnail to modal, tab indicators, reorder.

// manual FLIP const snap = jump.capture(el) doTheDOMChange() jump.flip(el, snap, { style: "spring" }) // shared element transition jump.snapshot(thumb, "hero") showModal() jump.shared(heroEl, "hero")
react

React bindings included. Tree-shakeable, SSR-safe, React 18 + 19. Every intent, as hooks and declarative components.

// hook — trigger: mount, visible, manual const { ref } = useJump("fade in slide up") const { ref } = useJump("enter from bottom", { trigger: "visible" }) // exit animations on unmount <Presence show={open} enter="fade in slide up" exit="fade out"> <Dialog /> </Presence> // shared element transition <Layout sharedId="card" style="bouncy"> <img src={thumb} /> </Layout> // drag-to-reorder <Reorder values={items} onReorder={setItems}> {items.map(item => ( <Reorder.Item value={item}>{item}</Reorder.Item> ))} </Reorder>
